# parallel_theory_3
## TASK 1
1. Используя std::thread, std::jthread или асинхронные задания, реализовать
многопоточную версию программы умножения матрицы на вектор с
параллельной инициализацией массивов.
2. Провести анализ маштабируемости.
3. Написать вывод о маштабируемости.

## TASK 2
Реализовать клиент-серверное приложение.
Сервер представляет из себя отдельный поток, который берет задачи из
очереди, решает их и сохраняет результат в контейнер. Реализовать в виде шаблона класса. 

У серевера обязательно должны быть следующие функции-члены:
1) void start() - запустить сервер;
2) void stop() - остановить сервер;
3) size_t add_task(task) - добавить задачу, возвращает id задачи;
4) T request_result(id_res) - возвращает результат типа T. Может быть блокирующим или не блокирующим.

Клиенты представляют из себя отдельные потоки, которые добаляют задачи в очередь, дожидаются результата и сохраняют в файл. 
Создать три клиента, каждый добавляет по N (5<N<10000) задач. Для каждого клиента использовать следующие задачи:
1) вычисление синуса, аргумент задавать рандомно;
2) вычисление квадратного корня, аргумент задавать рандомно;
3) возведение числа в степень, аргументы задавать рандомно.
